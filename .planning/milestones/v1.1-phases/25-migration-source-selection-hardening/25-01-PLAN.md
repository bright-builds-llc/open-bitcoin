# Phase 25 Plan 01: Thread Explicit Source Paths Into Detection

## Objective

Allow `migrate plan` to reuse the existing read-only detector for explicit
custom-location source datadirs by seeding the requested source path into the
migration command's detection roots.

## Scope

- Operator runtime detection flow for migration only
- End-to-end binary regression for a custom source datadir outside default roots

## Tasks

1. Add a runtime helper that chooses command-specific detection evidence.
2. For migration only, append the explicit `--source-datadir` to the detector's

   default home-based roots and verify the plan stays dry-run and secret-safe.
